# `@zatgo/ui`

Shared React UI for Next.js and Electron.

- **shadcn/ui**-style primitives live here (Radix + Tailwind tokens from `@zatgo/config`).
- Composites: `AppShellLayout`, `ErpnextLoginCard`, `PageHeader`, `DataTable`, `FormDialog`, `SearchField`, `EmptyState` / `LoadingState` / `ErrorState`, `StatCard`, `CommandPalette`, `createThemeStore`.

```bash
# From repo root
pnpm --filter @zatgo/ui typecheck
```
